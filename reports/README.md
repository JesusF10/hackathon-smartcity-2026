# Reportes y Análisis

Documentación generada a partir del análisis de datos para la toma de decisiones.

- `reporte_hermosillo_2024.pdf`: Informe detallado de siniestralidad, puntos críticos y factores de riesgo en Hermosillo.
- `reporte_hermosillo_2024.typ`: Archivo fuente en Typst para la generación del PDF.

### Generación de Reportes
Para compilar el reporte nuevamente:
```bash
typst compile reports/reporte_hermosillo_2024.typ
```
