# Datos Procesados - Hermosillo

Este directorio contiene los datasets específicos para el municipio de Hermosillo, Sonora (EDO 26, MPIO 30).

## Contenido
- `processed/`:
    - `hermosillo_maestro_2020_2024/`: Shapefile combinado de los últimos 5 años (47,744 registros).
    - `hermosillo_fatales_2020_2024/`: Puntos de accidentes con fallecidos (225 registros). Insumo principal para la campaña "Estrellas Negras".
    - `hermosillo_accidentes_todos_2024.csv`: Datos en formato tabular para análisis rápido.

## Licencia de Datos
Basado en datos de **INEGI - Accidentes de Tránsito en Zonas Urbanas y Suburbanas (ATUS)**.
